
# ERC20 Token Backend Application


#### Prerequisite
```
npm i
```
or
```
yarn
```
##### Enroll admin & register user
- Enroll **admin**
```
cd app/scripts && node enrollAdmin.js
```
- Register **user**
```
node registerUser.js && cd ...
```
##### Start
### Start with admin account for mint token
- Only **admin** with MSP **IntageMSP** can mint token
```
npm start admin

```
or
```
yarn start admin
```
